# Welcome to Kegworks
A user-friendly tool used to make wine wrapped ports of Windows software for macOS.

<br>

> [!NOTE]
> How to install Kegworks using [homebrew](https://brew.sh/)
> ```
> brew install --cask --no-quarantine gcenx/wine/kegworks
> ```
